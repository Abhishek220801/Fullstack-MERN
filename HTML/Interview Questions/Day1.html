Interview Questions Day-1

Q1. Difference between div and span?

Div tag - mainly use to create divisions and its a block level element and always starts with new line
Span tag- Mainly use to merge the row/column and its a inline element and does not alter document flow

Q2. Difference between class and IDs

Class is used to define the class for multiple items to group them together for certain purposes
ID - it should be unique for all elements and there will exist a single element with that ID in the whole document.

Q3. What are self closing tags?

Self Closing tags are those tags which does not need any closing tag to be complete as an element. They are itself the open and closing tags.
For example - < br/>, < input/>, < hr/>, < img/>, etc.

Q4. How many types of headings are there? How to decide which to use where?

Mainly we do have 6 types of headings from h1 to h6, on the basis of the importance/size of text lines, we use h1 being the strongest and biggest (most important heading) to h6 being the smallest (also the least important & least used)

Q5. Difference between Block and Inline elements?

Block level elements are elements which take 100% of its width of line and starts with new line always.
Inline level elements are those elements which take the width of line as per the requirement of text/content and start in same line.

Q6. Difference between em and strong?

< em> is abbreviation for emphasize and is used to provide stress/emphasis on the text or content inside the element and it always in italic.
< strong> is used to denote importance to the text or content inside the element and it is always in bold text.

Q7. What are attributes?

HTML attributes provide additional information about elements within an HTML document. Every HTML element can have attributes.
for example - style, height, width, src, etc.
