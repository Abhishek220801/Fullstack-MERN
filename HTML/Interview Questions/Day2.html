Interview Questions Day 2

Q1. Difference between HTML tags and elements?

HTML tags define the structural elements of a document, like < h1> for headings. Elements comprise both an opening tag, content, and a closing tag, enclosed within < and > symbols. These tags delineate the beginning and end of an HTML element.

Q2. Explain br tag

The < br> tag is self-closing, which means it doesn't need a corresponding closing tag. Web browsers do not inherently understand line breaks or paragraph structure, making the < br> tag essential for defining manual line breaks in HTML.

Q3. Explain how many different types of lists we have in html.

In HTML, there are two main types of lists that you are used to organize content:

  1. Ordered lists:
    • Ordered lists are used when the sequence of items is important, such as steps in a recipe or items in a numbered list.
    • Each item in an ordered list is marked with a number or another type of ordered marker (such as letters or Roman numerals).

  2. Unordered lists:
    • Unordered lists are used when the order of items is not important, such as a list of features or bullet points.
    • Each item in an unordered list is typically marked with a bullet point (or another type of unordered marker like a square or circle).

Q4. What are semantic tags?

Semantic HTML tags are tags that define the meaning of the content they contain.
for example, tags like header tag, article tag, and footer tag are semantic HTML tags. They clearly indicate the role of the content they contain.
